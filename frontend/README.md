# 프론트엔드 (Next.js)

ThinkingEngine 데이터 생성기의 웹 UI입니다.

## 기능

- Excel 스키마 파일 선택
- 시나리오, DAU, 날짜 범위 등 설정
- API Keys 및 ThinkingEngine 설정
- 데이터 생성 진행 상태 표시
- 통계 및 차트 미리보기
- LogBus2 업로드 제어

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS (예정)
- Recharts (차트, 예정)
